markdown的基本语法

  • markdown

1、简介

markdown是一种纯文本格式的标记语言。

2、标题

在文本前面添加#来表示。一个#是一级标题，两个#是二级标题，依次类推。

3、字体

  • 加粗
    要加粗的文本左右分别用两个*号包起来
  • 斜体
    要倾斜的文本左右分别用一个*号包起来
  • 斜体加粗
    要倾斜和加粗的文本左右分别用三个*号包起来
  • 删除线
    要加删除线的文本左右分别用两个~~号包起来

4、引用

在引用的文字前添加>即可，效果如下

这是引用的文字

5、分割线

三个或三个以上的-或*都可以

6、图片

![图片alt] (图片地址)

图片alt就是显示在图片下面的文字，相当于对图片内容的解释。
图片title是图片的标题，当鼠标移到图片上时显示的内容。title可加可不加

7、列表

  • 无序列表

无序列表使用- + *任何一种都可以；

注意：- + *跟内容之间都要有一个空格

  • 有序列表

有序列表使用数字加点即可

+列表嵌套
上一级和下一级之间敲三个空格即可

8、表格

语法：

姓名 技能 排行
刘备 哭 大哥
关羽 打 二哥

注意：
第二行分割表头和内容。
-有一个就行，为了对齐，多加了几个
文字默认居左
-两边加：表示文字居中
-右边加：表示文字居右

9、代码

  • 单行代码

代码之间分别用一个反引号包起来；效果如下：

create database hero

  • 代码块

10、删除线

文本两端加上两个～～即可，效果如下
删除我

11、链接

链接文字都是用方括号来标记:
简书

12、强调

两端各用一个*或者_包起来是斜体，各用两个包起来是粗体

single aster

single aster

13、流程图

MIU的markdown不支持流程图；mermaid的一般用法如下：

mermaid
graph LR
start[开始] –> input[输入A,B,C]
input –> conditionA{A是否大于B}
conditionA – Yes –> conditionC{A是否大于C}
conditionA – No –> conditionB{B是否大于C}
conditionC – Yes –> printA{输出A}
conditionC – No –> printC{输出C}
conditionB – Yes –> printB{输出B}
conditionB – No –> printC{输出C}
printA –> stop[结束]
printC –> stop
printB –>stop
